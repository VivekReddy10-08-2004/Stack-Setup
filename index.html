Zero-friction setup

Install Once. Start Coding.

Stack Setup installs your development tools with default settings so users can go from a clean machine to writing code in minutes.

1. Download and extract

Get Stack Setup and extract the folder anywhere. Then open a terminal in that folder.

https://github.com/VivekReddy10-08-2004/Stack-Setup/archive/refs/heads/main.zip Download
Required before setup

Prerequisite: Install Latest Python

Install the latest Python release first, then come back and run the setup command below.

https://www.python.org/downloads/ Download Latest Python

2. Run default install

No profile selection needed. The installer applies safe defaults for tools, VS Code setup, and starter projects.

python src/installer.py setup
Detecting OS... Python 3.10+ Windows · macOS · Linux

3. Open VS Code and build

Tools installed VS Code, language runtimes, and build tools are installed through your system package manager.
Editor ready Extensions and practical defaults are configured so formatting and linting work out of the box.
Projects generated Starter code is created so users can run and edit code immediately after setup.

Optional: preview install steps

If you want to inspect what will be installed before running setup, use dry-run mode.

python src/installer.py setup --dry-run

On Linux, some package installs may request sudo permission. VS Code extension setup requires the code command to be available in PATH.
